#### GET /categories
- Function:
    - Returns a list of categories, number of total categories. 
- Sample: 'curl http://127.0.0.1:5000/categories'
- Response:

```   "categories": {
    "1": "Science",
    "2": "Art",
    "3": "Geography",
    "4": "History",
    "5": "Entertainment",
    "6": "Sports"
  },
  "success": true,
  "total_categories": 6
}

```

#### POST /questions{search}
- Function:
    - Search for questions based on the specified search term.
    - Returns all the qualified questions
- Sample: curl http://127.0.0.1:5000/questions
- Json format: { "search": "Peanut"} 
- Response:
``` {
     "questions": [
        {
         "answer": "George Washington Carver",
         "category": 4,
         "difficulty": 2,
         "id": 12,
         "question": "Who invented Peanut Butter?"
        },  
     ],
     "totalQuestions": 1
   } 

```
